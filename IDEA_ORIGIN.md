# **VSCode quick timer**



* ###### A tools enabling the programmer to had a timer start or automatically initialized at the start of typing in the active tab in vscode editor



#### 

#### **Purpose:**

##### 	

###### 	To provide an ability to the user to track how much time he spent in that specific active tab.

###### 



#### **Idea Origin:**

	

###### 	I come up with this tools, to solve my problem from manually going to the clock application just to start a timer to check how much time i spent solving a competitive programming problem using python. While solving problem for short amount of time as possible it enhances my problem solving skill at tight amount of time.
